SookKai (สุขกาย) เป็นเว็บแอปพลิเคชัน (Web App) ที่ช่วยแนะนำวิธีการนวดกดจุดเพื่อบรรเทาอาการเจ็บป่วยทั่วไป เช่น ปวดหัว, ปวดคอ/บ่า/ไหล่ (Office Syndrome), นอนไม่หลับ, และปวดประจำเดือน โดยมีระบบนำทางทีละขั้นตอน พร้อมเสียงบรรยายและดนตรีบำบัด

## ✨ ฟีเจอร์หลัก (Key Features)

* **Symptom Menu:** เลือกอาการที่ต้องการรักษาได้หลากหลาย (ปวดหัว, นอนไม่หลับ, ภูมิแพ้ ฯลฯ)
* **Step-by-Step Guide:** ระบบนำทางทีละจุด พร้อมรูปภาพประกอบตำแหน่งที่ถูกต้อง
* **Audio Coach:** มีเสียง AI อ่านคำแนะนำและจังหวะการหายใจ (Text-to-Speech)
* **Relaxing Music:** ระบบเล่นเพลงผ่อนคลายอัตโนมัติ (รองรับไฟล์ Local และ Online)
* **Smart Timer:** ตัวจับเวลาถอยหลังสำหรับแต่ละจุด เพื่อให้ได้ผลลัพธ์ที่ดีที่สุด
* **Gamification:** ระบบสะสมแต้มสุขภาพ (Streak) นับจำนวนวันที่ดูแลตัวเองต่อเนื่อง
* **Onboarding:** หน้าต้อนรับและคำแนะนำความปลอดภัยก่อนใช้งาน (3 กฎเหล็ก)
* **Responsive Design:** รองรับการใช้งานทั้งบนมือถือและคอมพิวเตอร์

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

* **HTML5** (Semantic Structure)
* **CSS3** (Modern Flexbox/Grid, Animations, Responsive)
* **JavaScript** (Vanilla JS - No Framework required)
* **Local Storage** (สำหรับบันทึกสถิติ Streak รายวัน)
